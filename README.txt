# WebGIS com Python, Streamlit e Folium

## Como rodar o projeto

1. Instale as dependências:
   ```
   pip install -r requirements.txt
   ```

2. Execute o aplicativo:
   ```
   streamlit run app.py
   ```

3. O mapa será aberto no navegador. Você pode alternar entre 3 mapas de fundo e exibir/ocultar o shapefile.

- O shapefile deve estar na pasta `Shapefile`.
- O zoom inicial foca na área do shapefile.
